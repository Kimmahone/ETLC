# 🦁 ETLC 동물 성격유형 검사 (ETLC Animal Personality Test)

> **"나를 이해하고, 타인을 존중하며, 함께 성장하는 자존감의 열쇠"**

ETLC 동물 성격유형 검사는 개인이 가진 기본적인 성향(Life Style)과 문제 해결 시 주로 사용하는 전략(Operate System)을 알아보는 웹 애플리케이션입니다. 

**누구나 자신을 돌아보고 타인과의 다름을 긍정적으로 수용할 수 있는** 내용으로 구성되었습니다.

<br>

## ✨ 주요 기능 (Features)

* **직관적인 UI/UX:** 부드러운 화면 전환과 모바일 환경을 고려한 반응형 디자인(Tailwind CSS)을 적용하여 누구나 쉽게 검사를 진행할 수 있습니다.
* **심층적인 결과 분석:** 48개의 문항(가중치 문항 포함)을 분석하여 사용자의 메인 성격(Life Style)과 보조 성격(Operate System)을 도출합니다.
* **맞춤형 피드백 제공:** 도출된 유형에 따라 '나의 강점', '나의 약점', 그리고 '성장을 위한 조언'을 상세히 제공합니다.
* **결과 PDF 다운로드:** 화면 캡처 방식의 한계를 넘어, UI 요소(버튼 등)를 숨기고 문서 형태의 깔끔한 여백을 적용한 고품질 PDF 다운로드 기능을 지원합니다.
* **검사 다시하기:** 페이지 새로고침 없이 즉시 검사를 초기화하고 다시 진행할 수 있습니다.

<br>

## 🐾 4가지 동물 성격 유형

검사 결과는 다음 4가지 동물 유형의 조합으로 나타납니다.

* 🦅 **독수리 (E):** 높은 곳에서 넓게 보는 시야를 가진 유형입니다. 이상적이고 완벽함을 추구하며 모든 것을 스스로 해결하려 합니다.
* 🐢 **거북이 (T):** 느리지만 묵묵히 자신의 길을 가는 유형입니다. 문제 해결보다는 갈등을 피하고 싶어 하는 경향이 있습니다.
* 🦁 **사자 (L):** 강력한 힘과 리더십을 상징하는 유형입니다. 목표를 세우고 주도적으로 이끌어가는 능력이 뛰어납니다.
* 🦎 **카멜레온 (C):** 주변 환경에 맞춰 자신의 색을 바꾸는 유형입니다. 외부의 평가에 민감하며, 다른 사람의 기대에 맞춰 행동하려는 경향이 있습니다.

<br>

## 🛠 사용 기술 (Tech Stack)

이 프로젝트는 별도의 백엔드 서버 없이 프론트엔드 기술만으로 가볍고 빠르게 동작하도록 설계되었습니다.

* **HTML5 / CSS3 / JavaScript (Vanilla)**
* **Tailwind CSS:** 빠르고 일관된 반응형 UI 스타일링을 위해 CDN 방식으로 사용
* **jsPDF & html2canvas:** 검사 결과 화면을 고품질 PDF 문서로 변환하고 다운로드하기 위한 라이브러리

<br>

## 🚀 실행 방법 (How to Run)

단일 HTML 파일로 구성되어 있어 복잡한 설치 과정이 필요 없습니다.

1. 이 저장소를 클론(Clone)하거나 `index.html` 파일을 다운로드합니다.
2. 다운로드한 `index.html` 파일을 크롬, 사파리, 엣지 등 최신 웹 브라우저에서 실행합니다.
3. 준비 활동(피하고 싶은 상자 고르기)을 시작으로 48개의 문항에 솔직하게 답합니다.
4. 결과를 확인하고 PDF로 저장해 보세요!
